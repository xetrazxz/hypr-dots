## Prerequisites
```bash
git clone https://aur.archlinux.org/yay.git ~/yay
cd ~/yay
makepkg -si
cd ~/
```


#Installation
```bash
git clone https://github.com/xetrazxz/hypr-dots.git ~/hypr-dots

chmod +x install.sh
./install.sh
```
